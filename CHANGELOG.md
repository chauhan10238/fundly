# Changelog

## DIOS 1.1.0 — Personal Portfolio Foundation

- Replaced fabricated demo holdings with the current Stake snapshot: VT, GLD, MLPX, TSM, AMD and INTC.

- Normalised tickers and improved fractional-share handling.
